Posteo 2.3 Fotografía + origami

Diseño molde en Grasshopper Rhino.

Luego de realizar un patron en Grasshopper y de cocinarlo, utilizando Rhino se fue acomodando cada una de las piezas para lograr obtener el molde final.

Proceso de armado del molde.

Comenzé haciendo el molde con papel para ver como sería este. El cual el resultado me gusto mucho por lo que comenzó el desafío de como lograr hacer un molde que soporte chocolate derretido el cual esta a alta temperatura y se deforme.

En el video podrán ver el proceso de cortado de las piezas para crear el molde. Luego de cortar las piezas estas fueron cubiertas en papel mantequilla con la finalidad de que el chocolate no se pegue al carton pierda. Finalmente 5 de las piezas fueron pegadas para crear la 1/2 del molde.

Si bien no fue un desafío facil, se intento varias veces lograr hacer el molde a base de prueba y error para llegar al mejor resultado posible. En la primera foto donde se ve un molde de aluminio dentro de un bowl, fue el primer intento, el cual no resistió ya que en su interior no era carton sino papel. La segunda imagen es la 1/2 del molde pero cometí el error de unir las piezas y después colocar el aluminio el cual debio alrevez.

Ingredientes.

los ingredientes utilizados fueron los sigueintes.

  1. Chocolate
  2. Frambuesa
  3. Arandanos
  4. Agua
  5. Azúcar flor
  6. Merengue powder

Proceso de producción.

Antes de llegar a un buen resultado siempre se esta permitido fallar, en este caso fallé, más de una vez pero asi es como se aprende. A continuación veremos algunas imágenes y video del proceso de producción del producto.

Como podemos observar en el video, el molde fue pintado con chocolate para lograr llegar a todos los lugares y esquinas posibles.
