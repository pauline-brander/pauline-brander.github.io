Posteo 3.1 Unidad 3

A continuación podrán observar el primer intento de armado de la propuesta para la Unidad 3. Donde se utilizó hojas de block blancas, tigera, hilo y aguja (debido a que no tenía corchetera)

Lo primero que se hizo fue comenzar a cortar papeles (hoja de block) y coser algunos lugares dando forma, después se realizo la fotogametría para finalmente llevarlo a Rhino7 y con el comando interpolateCrv realizar el modelo junto con el comando Patch.
