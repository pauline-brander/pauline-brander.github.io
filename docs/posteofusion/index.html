Posteo 2.1 Fusion 360

Aprender a utilizar un software nuevo siempre es emocionante, pero también es desafiante, a continuación podrán ver alguno de los resultados obtenidos siguiendo los pasos de los tutoriales de Fusion 360.

Sketch y Revolve

El primer tutorial corresponde a “Sketch y Revolve” donde se enseña como comenzar un dibujo para el objeto que se desea diseñar. El diseño que realicé es muy similar al del video debido a que aún me encuentro aprendiendo como utulizar el software y todos sus componentes.

Superficie Loft

El segundo tutorial corresponde a “Superficie Loft” donde aprendimos como desde curvas podemos realizar un molde, lo cual en un futuro puede ser aplicado para distintos objetos. Algo aprendido durante el proceso de este objeto es que hay que tener en cuenta que al realizar un “fillet” hay que tener en consideración todas las curvas para que se pueda llevar a cabo, de lo contrario no se podrá realizar el “fillet”.

Form Molde de 2 Piezas y Draft

El tercer y cuarto tutorial, tiene el proposito de mostrarnos como realizar un molde de dos piezas el cual nos servira como guía el desarrollo de la unidad “Bits & Bites”, y familiarizarse con distintas aplicaciónes que tiene el software, además de enseñar que hay que considerar el desmolde el cual se puede ver utilizando “Section Analysis”
